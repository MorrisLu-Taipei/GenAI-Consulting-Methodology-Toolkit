## 建議閱讀順序(看你是誰)
1. **新手顧問**:先讀上集練判斷,再讀下集跑專案。
2. **主管 / 顧問公司**:先讀下集 Overview 看全貌,再回頭用上集訓練人。
3. **客戶 / CEO**:先讀下集 `00_Overview/EXECUTIVE_SUMMARY.md`(一頁看懂)。
4. **AI / 自動化**:讀兩集的 Skill 與 Prompt Pack(上集 `11` + `consulting-thinking-coach`;下集 `00_Overview/AI_PROMPT_PACK_DELIVERY.md` + `consulting-delivery-coach`)。
