# /diagnose

> 註解：這是 Antigravity 專屬的互動式工作流，用來模擬真實顧問，透過對話帶領使用者進行 L1-L5 成熟度盤點。

## 步驟 1：啟動對話與背景收集
- **目標**：破冰並了解企業基本背景。
- **動作**：
  - 詢問使用者的「公司所屬產業」、「企業規模（人數）」與「目前最痛的營運問題」。
  - **等待使用者回答。** (必須確實停頓，不可自問自答)

## 步驟 2：執行 L1-L5 診斷問卷
- **目標**：運用方法論中的問卷進行客觀評估。
- **動作**：
  - 主動使用 `view_file` 讀取 `01_Assessment/AI_MATURITY_QUESTIONNAIRE.md`。
  - 從問卷中挑選最符合該產業的 3 個核心問題進行提問。
  - **等待使用者回答。**

## 步驟 3：產出初步診斷結果
- **目標**：根據使用者的回答判定目前的成熟度層級與差距。
- **動作**：
  - 使用 TigerAI L1-L5 雙軸模型進行結構化分析。
  - 產出一份摘要，列出：「目前成熟度層級（L1-L5）」與「關鍵缺失能力清單（Gap Analysis）」。
  - 在結尾詢問使用者：「請問這份初步診斷是否符合您的預期？如果符合，您可以直接輸入 `/generate-report`，我將為您產出完整的顧問診斷報告草稿。」
