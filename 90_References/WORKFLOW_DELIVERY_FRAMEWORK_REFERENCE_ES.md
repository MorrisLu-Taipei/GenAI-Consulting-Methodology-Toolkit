## 2. Qué es

Un **framework de estándar de entrega profesional** para consultores / agencias de n8n automation, cubriendo todo el ciclo de negocio:

- **Ciclo de vida de tres fases Sales → Delivery → Support**
- **7 Pillars**: client hosts, client pays, security first, test thoroughly, document fully, clean handover, clear scope
- **SOPs de 7 roles**: lead gen, sales rep, closer, PM, technical lead, developer, client
- **200+ checklists**: pre-project, security, QA, handover, offboarding
- **Business document templates**: SOW, contratos, facturas, propuestas, handover, change orders, email
- **Guías**: onboarding, security, testing, delivery, maintenance, offboarding, pricing, risk, workflow standards, troubleshooting
