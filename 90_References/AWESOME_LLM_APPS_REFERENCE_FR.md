## 2. Ce que c'est

Un « livre de recettes d'apps LLM exécutables » — 100+ templates de départ construits à la main, testés sur 13 catégories : Starter Agents, Advanced Agents, Game-Playing Agents, Multi-agent Teams, Voice Agents, MCP Agents, RAG, Agent Skills, Memory Apps, Chat with X, Optimization, Fine-tuning et Framework Crash Courses.

## 4. Résumé de la licence

awesome-llm-apps est sous **Apache License 2.0** — permettant fork libre, customisation et usage commercial. Cette méthodologie **ne redistribue pas** son code source — elle cite uniquement noms de cas et catégories comme index dans `LLM_APPS_CASE_INDEX.md`. Les apprenants souhaitant réellement exécuter une app doivent l'obtenir du repo upstream et se conformer à ses termes Apache 2.0 (préserver les avis de copyright et licence).
