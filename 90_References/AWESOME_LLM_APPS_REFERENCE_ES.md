## 2. Qué es

Un "libro de recetas de apps LLM ejecutables" — 100+ templates de inicio hechos a mano, probados en 13 categorías: Starter Agents, Advanced Agents, Game-Playing Agents, Multi-agent Teams, Voice Agents, MCP Agents, RAG, Agent Skills, Memory Apps, Chat with X, Optimization, Fine-tuning y Framework Crash Courses.

## 4. Resumen de licencia

awesome-llm-apps está bajo **Apache License 2.0** — permitiendo fork libre, customización y uso comercial. Esta metodología **no redistribuye** su código fuente — solo cita nombres de casos y categorías como índice en `LLM_APPS_CASE_INDEX.md`. Los aprendices que deseen realmente ejecutar una app deben obtenerla del repo upstream y cumplir con sus términos Apache 2.0 (preservando los avisos de copyright y licencia).
