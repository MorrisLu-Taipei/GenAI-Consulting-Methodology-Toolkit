## 2. TigerAI-n8n-Skill-Pack 是什麼 / What is TigerAI-n8n-Skill-Pack

它讓使用者用 **自然語言描述** 生成企業級 n8n workflow，而非手動設定節點。系統把「便利貼」式的白話意圖轉成完整、有文件的 workflow JSON，可直接部署。

It lets users generate enterprise-grade n8n workflows from **natural-language descriptions** rather than manual node configuration. The system transforms "sticky-note" plain-language intent into complete, documented workflow JSON ready for deployment.

### 三層結構 / Three-Layer Structure

`黃色便利貼意圖（intent）` + `藍色便利貼註記（notes）` + `workflow nodes` — 使用者寫意圖 → AI 生成三層 JSON → 透過 n8n API 部署。

### 三種使用模式 / Three Usage Modes

1. **Cookbook copy** — 從 8 個現成 recipe 模板挑選
2. **Q&A mode** — AI 引導使用者完成需求蒐集
3. **Example finder** — 從 2,061 reference workflows 找相似實作

### 安裝 / Installation

`bash install.sh`（Unix）或 `install.ps1`（Windows）。

## 3. 為什麼放在 L3 下半段 / Why It Belongs in the L3 Second Half

L3 課程依「先觀念、後生成」切成上下半段：

- **L3 上半段**（§5.1 Foundation + §5.2 Builder）：學員**親手**建過 Trigger / Node / AI / Human Gate / Error Handling，理解 workflow 結構與治理。
- **L3 下半段**（§5.5）：在已懂結構的基礎上，用 Antigravity + TigerAI-n8n-Skill-Pack 把自然語言意圖直接生成 workflow JSON，並學會**審查**生成結果。

The L3 course is split into halves on a "concepts first, generation second" principle:

- **L3 first half** (§5.1 Foundation + §5.2 Builder): learners **hand-build** Trigger / Node / AI / Human Gate / Error Handling, understanding workflow structure and governance.
- **L3 second half** (§5.5): on that foundation, use Antigravity + TigerAI-n8n-Skill-Pack to generate workflow JSON directly from natural-language intent, and learn to **review** the generated result.

> **核心原則 / Core principle：Skill Pack 是加速器，不是替代理解。** 學員若沒先在上半段親手建過 workflow，就無法判斷生成結果是否正確、安全、可維運。
> **The Skill Pack is an accelerator, not a substitute for understanding.** Without hand-building workflows in the first half, learners cannot judge whether generated results are correct, safe, or maintainable.
