## 2. What is TigerAI-n8n-Skill-Pack

It lets users generate enterprise-grade n8n workflows from **natural-language descriptions** rather than manual node configuration. The system transforms "sticky-note" plain-language intent into complete, documented workflow JSON ready for deployment.

### Three-Layer Structure

`yellow sticky-note intent` + `blue sticky-note notes` + `workflow nodes` — the user writes intent → AI generates the three-layer JSON → deploy via the n8n API.

### Three Usage Modes

1. **Cookbook copy** — select from the 8 pre-built recipe templates
2. **Q&A mode** — AI guides the user through requirements gathering
3. **Example finder** — discover similar implementations among the 2,061 reference workflows

### Installation

`bash install.sh` (Unix) or `install.ps1` (Windows).

## 3. Why It Belongs in the L3 Second Half

The L3 course is split into halves on a "concepts first, generation second" principle:

- **L3 first half** (§5.1 Foundation + §5.2 Builder): learners **hand-build** Trigger / Node / AI / Human Gate / Error Handling, understanding workflow structure and governance.
- **L3 second half** (§5.5): on that foundation, use Antigravity + TigerAI-n8n-Skill-Pack to generate workflow JSON directly from natural-language intent, and learn to **review** the generated result.

> **Core principle: the Skill Pack is an accelerator, not a substitute for understanding.** Without hand-building workflows in the first half, learners cannot judge whether generated results are correct, safe, or maintainable.
