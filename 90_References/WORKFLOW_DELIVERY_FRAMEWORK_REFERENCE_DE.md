## 2. Was es ist

Ein **professionelles Delivery-Standard-Framework** für n8n-Automatisierungs-Berater / Agenturen, das den gesamten Business-Zyklus abdeckt:

- **Sales → Delivery → Support Drei-Phasen-Lebenszyklus**
- **7 Pillars**: Client hosts, Client pays, Security first, Test thoroughly, Document fully, Clean handover, Clear scope
- **7-Rollen SOPs**: Lead Gen, Sales Rep, Closer, PM, Technical Lead, Developer, Client
- **200+ Checklisten**: Pre-Project, Security, QA, Handover, Offboarding
- **Business Document Templates**: SOW, Verträge, Rechnungen, Proposals, Handover, Change Orders, Email
- **Guides**: Onboarding, Security, Testing, Delivery, Maintenance, Offboarding, Pricing, Risk, Workflow Standards, Troubleshooting
