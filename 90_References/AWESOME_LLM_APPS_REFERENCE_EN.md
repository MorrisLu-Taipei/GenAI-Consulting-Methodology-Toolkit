## 2. What It Is

A "cookbook of runnable LLM apps" — 100+ hand-built, tested starter templates across 13 categories: Starter Agents, Advanced Agents, Game-Playing Agents, Multi-agent Teams, Voice Agents, MCP Agents, RAG, Agent Skills, Memory Apps, Chat with X, Optimization, Fine-tuning, and Framework Crash Courses.
