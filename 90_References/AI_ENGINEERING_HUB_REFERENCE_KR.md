**코드 우선 AI 엔지니어링 학습 허브** — OCR & Vision, 채팅 인터페이스, RAG 시스템, agents & multi-agent, voice & audio, fine-tuning, 고급 인프라 (MCP), 모델 비교에 걸친 93+ 프로덕션 준비 프로젝트. 단순한 링크 디렉토리와 달리, 실행 가능한 코드 구현과 구조화된 점진적 학습 경로를 제공.

## 3. 무엇을 인용했나

| 범위 | 처리 |
| --- | --- |
| **대표적 프로젝트 이름과 카테고리** | `LLM_APPS_CASE_INDEX.md` 에서 인덱스 항목으로 인용 (source tag H) |

> 인덱스는 ai-engineering-hub 의 이름 있는, 대표적 프로젝트를 다룸; 93+ 전체 리스트는 상류 repo 참조 — 시간이 지나면서 인덱스에 추가 가능.

## 4. 라이선스 요약

ai-engineering-hub 는 **MIT License** 하 — 프로프라이어터리 제품의 일부로 포함하여 자유 사용, 수정, 재배포, 상업적 사용을 허용. 유일한 조건은 소스 재배포 시 MIT 저작권 통지와 라이선스 텍스트를 보존하는 것.

본 방법론은 소스 코드를 **재배포하지 않음** — `LLM_APPS_CASE_INDEX.md` 에서 대표적 프로젝트 이름과 카테고리를 인덱스로만 인용. 실제로 프로젝트를 실행하고 싶은 학습자는 상류 repo 에서 얻고, 그 MIT 조건을 준수해야 함.
