## 2. Ce que c'est

Un **framework de standard de livraison professionnel** pour consultants / agences n8n automation, couvrant tout le cycle business :

- **Cycle de vie trois phases Sales → Delivery → Support**
- **7 Pillars** : client hosts, client pays, security first, test thoroughly, document fully, clean handover, clear scope
- **SOPs 7-rôles** : lead gen, sales rep, closer, PM, technical lead, developer, client
- **200+ checklists** : pre-project, security, QA, handover, offboarding
- **Business document templates** : SOW, contrats, factures, propositions, handover, change orders, email
- **Guides** : onboarding, security, testing, delivery, maintenance, offboarding, pricing, risk, workflow standards, troubleshooting
