## 2. 그것이 무엇인가

n8n 자동화 컨설턴트 / 에이전시용 **전문 딜리버리 표준 프레임워크**, 전체 비즈니스 사이클 커버:

- **Sales → Delivery → Support 3 단계 생애주기**
- **7 Pillars**: client hosts, client pays, security first, test thoroughly, document fully, clean handover, clear scope
- **7 역할 SOPs**: lead gen, sales rep, closer, PM, technical lead, developer, client
- **200+ 체크리스트**: pre-project, security, QA, handover, offboarding
- **Business document templates**: SOW, 계약, 청구서, 제안, handover, change orders, email
- **가이드**: onboarding, security, testing, delivery, maintenance, offboarding, pricing, risk, workflow standards, troubleshooting
