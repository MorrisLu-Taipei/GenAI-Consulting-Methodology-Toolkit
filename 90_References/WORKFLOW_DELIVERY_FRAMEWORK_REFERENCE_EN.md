## 2. What It Is

A **professional delivery-standard framework** for n8n automation consultants / agencies, covering the entire business cycle:

- **Sales → Delivery → Support three-phase lifecycle**
- **7 Pillars**: client hosts, client pays, security first, test thoroughly, document fully, clean handover, clear scope
- **7-role SOPs**: lead gen, sales rep, closer, PM, technical lead, developer, client
- **200+ checklists**: pre-project, security, QA, handover, offboarding
- **Business document templates**: SOW, contracts, invoices, proposals, handover, change orders, email
- **Guides**: onboarding, security, testing, delivery, maintenance, offboarding, pricing, risk, workflow standards, troubleshooting
