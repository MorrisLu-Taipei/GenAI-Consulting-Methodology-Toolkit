## 2. それは何か

n8n オートメーションコンサルタント / エージェンシー向け**プロフェッショナルデリバリー標準フレームワーク**、ビジネスサイクル全体をカバー：

- **Sales → Delivery → Support 3 フェーズライフサイクル**
- **7 Pillars**：client hosts、client pays、security first、test thoroughly、document fully、clean handover、clear scope
- **7 役割 SOPs**：lead gen、sales rep、closer、PM、technical lead、developer、client
- **200+ チェックリスト**：pre-project、security、QA、handover、offboarding
- **Business document templates**：SOW、契約、請求書、提案、handover、change orders、email
- **ガイド**：onboarding、security、testing、delivery、maintenance、offboarding、pricing、risk、workflow standards、troubleshooting
