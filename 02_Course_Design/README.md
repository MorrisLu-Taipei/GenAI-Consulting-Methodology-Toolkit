# 02 Course Design

本目錄放課程設計依據與課程模組。

- `COURSE_REQUIREMENTS_AND_COMPANY_PROFILE.md`：L1-L5 課程需求清單、公司屬性調查、雲 AI / Hybrid / 全地端部署考量。
- `COURSE_MODULE_MATRIX.md`：L1-L5 課程大綱、實作練習、課後產出物與課程包裝。
- `L1_L5_COMPLETE_COURSE_PLAN.md`：L1-L5 完整開課規劃，包含課程目標、上課內容、實作、課後作業、完成標準與 Stage Gate。
- `L4_HERMES_AGENT_COURSE_PLAN.md`：參考 Hermes `starter-kit-v2` 的 L4 深度課程，包含 Wiki 記憶、五個核心 skills、ingest/query/update、briefing/discovery、evidence 與 Gate 4A-4E。
