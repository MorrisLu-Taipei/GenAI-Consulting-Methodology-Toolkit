# 01 Assessment

> 🌐 中文版本 / Chinese version: [README.md](README.md)

This directory contains the AI maturity diagnostic toolkit.

- `AI_MATURITY_QUESTIONNAIRE.md`: 10-item, 25-item, and 50-item questionnaires.
- `AI_MATURITY_SCORING_MODEL.md`: L1-L5 scoring logic, maturity determination, deployment mode, and course-ratio recommendation.
- `AI_MATURITY_DELIVERABLES_AND_EVIDENCE_MATRIX.md`: Per-stage Definition of Done, Deliverables, Evidence, Owner, Stage Gate, Fail Condition, and Next Level Entry Criteria for L1-L5.
- `FILLABLE_QUESTIONNAIRE.md`: Fillable-form specs for the 10/25/50-item versions (Google Form / Tally / Notion Form rendering, skip logic, post-submit flow).
- `COMPANY_PROFILE_QUESTIONNAIRE.md`: 35-question company-profile form that emits a JSON Profile Bundle, with derivation rules for deployment mode, course ratio, and PoC scenarios.
- `AI_DIAGNOSIS_SHEETS_SCHEMA.md`: Diagnosis-automation schemas — Google Sheets scoring sheet, Notion 4-database structure, and an n8n 13-node auto-diagnosis flow.
