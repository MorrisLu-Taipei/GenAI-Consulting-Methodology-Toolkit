# 01 Assessment

本目錄放 AI 成熟度診斷工具。

- `AI_MATURITY_QUESTIONNAIRE.md`：10 題、25 題、50 題問卷。
- `AI_MATURITY_SCORING_MODEL.md`：L1-L5 評分邏輯、成熟度判定、部署模式與課程比例推薦。
- `AI_MATURITY_DELIVERABLES_AND_EVIDENCE_MATRIX.md`：L1-L5 每階段 Definition of Done、Deliverables、Evidence、Owner、Stage Gate、Fail Condition 與 Next Level Entry Criteria。
- `FILLABLE_QUESTIONNAIRE.md`：10/25/50 題版可填寫問卷規格（Google Form / Tally / Notion Form 渲染、skip logic、提交後流程）。
- `COMPANY_PROFILE_QUESTIONNAIRE.md`：35 題公司屬性問卷，輸出 JSON Profile Bundle，並含部署模式 / 課程比例 / PoC 場景之推導規則。
- `AI_DIAGNOSIS_SHEETS_SCHEMA.md`：診斷自動化 schema — Google Sheets 計分表、Notion 4-database 結構、n8n 13-node 自動診斷流程。
