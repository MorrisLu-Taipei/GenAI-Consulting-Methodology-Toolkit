# 04 Scenarios

本目錄放客戶情境故事。

- `CUSTOMER_SCENARIO_LIBRARY.md`：CEO、COO、IT、HR 與業務、客服、行銷、營運、財務、人資、IT 部門故事。
- `CASE_WRITING_STANDARD.md`：案例撰寫標準，規定 L1-L5 Input / Process / Output / Evidence 與可驗證交付物。
- `CASE_CONTROL_TABLES.md`：案例控制表格，可直接複製填寫評估活動、L1-L5 IPOE、Evidence、Stage Gate、風險治理與 Deliverables 驗收。
- `SAMPLE_CLIENT_CASE_MANUFACTURING.md`：研發製造業完整示範案例，從問卷、課程比例、八階段診斷到 Roadmap。
- `SAMPLE_CLIENT_CASE_HOSPITAL.md`：醫院 / 醫療機構完整示範案例，強調高敏感資料、全地端 / 嚴格 Hybrid、人工審核與稽核。
