# 04 Scenarios

本目錄放客戶情境故事、產業場景、完整示範案例與 L5 實作 walkthrough。

## 情境與標準 / Scenarios & Standards

- `CUSTOMER_SCENARIO_LIBRARY.md`：CEO、COO、IT、HR 與業務、客服、行銷、營運、財務、人資、IT 部門故事。
- `CASE_WRITING_STANDARD.md`：案例撰寫標準，規定 L1-L5 Input / Process / Output / Evidence 與可驗證交付物。
- `CASE_CONTROL_TABLES.md`：案例控制表格，可直接複製填寫評估活動、L1-L5 IPOE、Evidence、Stage Gate、風險治理與 Deliverables 驗收。
- `INDUSTRY_SCENARIOS.md`：零售 / 教育 / 物流 / 軟體 / 專業服務 5 產業推薦場景，每產業含簡介、L1-L5 基線、Top 10 場景、風險治理、30 天 Quick Win、Anti-Patterns。

## 完整示範案例 / Full Sample Client Cases

> 所有客戶皆以代號表示（非真實公司名）。

- `SAMPLE_CLIENT_CASE_MANUFACTURING.md`：研發製造業完整示範案例。
- `SAMPLE_CLIENT_CASE_HOSPITAL.md`：醫院 / 醫療機構完整示範案例，強調高敏感資料、全地端 / 嚴格 Hybrid、人工審核與稽核。
- `SAMPLE_CLIENT_CASE_MARKETING_AGENCY.md`：行銷代理商案例（代號：代理商 M）。
- `SAMPLE_CLIENT_CASE_B2B_SALES.md`：B2B 工業設備商案例（代號：工業設備商 B），聚焦 RFP 回覆、CRM 治理、L5 Pre-RFP Team。
- `SAMPLE_CLIENT_CASE_FINANCIAL.md`：金融業案例（代號：區域銀行 F），全地端、法遵+資安雙審。
- `SAMPLE_CLIENT_CASE_GOVERNMENT.md`：政府機關案例（代號：市政府數位局 G），全地端、政風+法制+資安三審。
- `SAMPLE_CLIENT_CASE_EDUCATION.md`：教育機構案例（代號：科技大學 E），Hybrid、學生個資地端、學術倫理。
