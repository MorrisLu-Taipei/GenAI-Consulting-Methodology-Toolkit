# 03 Consulting Report

This directory contains the consulting diagnostic report template and the consulting toolkit.

- `CONSULTING_REPORT_TEMPLATE.md`: AI transformation diagnostic report Markdown template, covering maturity, As-Is, gap analysis, To-Be, Roadmap, ROI, and SOW recommendations.
- `CONSULTING_TOOLKIT_TEMPLATES.md`: Ready-to-use toolkit templates for the eight-stage consulting methodology — a 40-question interview bank, AI/systems inventories, As-Is Map, Vision Workshop, industry benchmark templates, Missing/Broken/Redundant table, Impact×Effort matrix, Problem Statement worksheet, 30/60/90 Roadmap, Gate 1-5 checklists, Risk Register, Skill/Workflow/Agent Map, permission/ROI/audit/ethics, and a 6-week engagement schedule.
