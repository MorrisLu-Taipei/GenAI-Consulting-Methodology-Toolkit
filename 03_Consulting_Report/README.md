# 03 Consulting Report

本目錄放顧問診斷報告模板與顧問工具表。

- `CONSULTING_REPORT_TEMPLATE.md`：AI 轉型診斷報告 Markdown 模板，包含成熟度、As-Is、差距、To-Be、Roadmap、ROI 與 SOW 建議。
- `CONSULTING_TOOLKIT_TEMPLATES.md`：八階段顧問診斷工具表（可直接使用）— 40 題訪談題庫、AI/系統盤點表、As-Is Map、Vision Workshop、產業標竿模板、Missing/Broken/Redundant 表、Impact×Effort 矩陣、Problem Statement worksheet、30/60/90 Roadmap、Gate 1-5 checklist、Risk Register、Skill/Workflow/Agent Map、權限/ROI/Audit/Ethics、6 週標準排程。
- `CONSULTING_FRAMEWORKS_LIBRARY.md`：經典顧問框架庫 — 50+ 個業界框架（MECE、Pyramid Principle、Porter's 5 Forces、SWOT、BCG Matrix、5 Whys、Business Model Canvas、WBS/RACI、NPV/IRR、Lean/Six Sigma 等），分 7 大類，附「框架選擇器」與「框架組合鏈」，每個框架對應到八階段。改寫自 [yoichiojima-2/consultant](https://github.com/yoichiojima-2/consultant) (MIT)，引用見 `../90_References/CONSULTANT_FRAMEWORKS_REFERENCE.md`。
- `REPORT_PRODUCTION_WORKFLOW.md`：顧問報告/簡報生產工作流 — 8 步把「問題→可交付報告/簡報」（定義邊界→Issue Tree+假設→Dummy Pages→逐頁迭代→PPT+Excel→Word→修訂），含 Dummy Page、依賴管理、7 頁面版型、Excel 證據軌、progressive disclosure，對應八階段與 3 套 deck outline。改寫自 [Kafurtan/mckinsey-consultant-skills](https://github.com/Kafurtan/mckinsey-consultant-skills) (MIT)，引用見 `../90_References/MCKINSEY_SKILLS_REFERENCE.md`。
