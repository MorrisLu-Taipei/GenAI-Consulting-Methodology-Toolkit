## 1. Hero Tagline

> **Enterprise AI transformation is not about adopting tools — it is about building verifiable maturity.**

Subtitle: Use the L1-L5 maturity model to evolve AI from individual usage to autonomous Agent Team collaboration.

## 2. Problem Statement

- **No alignment on AI usage:** shared accounts, password sharing, leaked sensitive data.
- **Tools without methodology:** ChatGPT / Copilot licenses bought, but no measurable outcomes.
- **Failed-to-scale RAG / Agent pilots:** PoCs that never become production workflows.

## 3. Three-Step Service Flow

1. **Diagnose** — 10/25/50-question AI maturity questionnaire + company-profile survey → objective L1-L5 score
2. **Build** — L1-L5 courses allocated by score; in-class production of verifiable assets (Prompt / Skill / Workflow / Agent / Team)
3. **Deliver** — eight-stage consulting diagnostic report + 30/60/90-day Roadmap + ROI and governance recommendations

`Diagnose → Build → Deliver` — 6 to 12 weeks per engagement.

## 4. L1-L5 Maturity Map (text representation)

| Level | Name | Tool | One-line value |
| --- | --- | --- | --- |
| **L1** | Chat AI | OpenWebUI | Org-wide safe AI usage; accumulate prompts and scenarios |
| **L2** | Skill AI | Antigravity / Claude Code / Codex | Package personal know-how into reusable Skills |
| **L3** | Workflow AI | n8n | Wire Skills into Gmail/Sheets/Notion/CRM/ERP as Workflows |
| **L4** | Auto Agentic AI | Hermes Agent | Wiki + Skill + Workflow + Gate = verifiable Agent |
| **L5** | Agentic Team AI | ClawTeam | Multiple specialist Agents form a Team for cross-functional tasks |

> Each layer's output is the next layer's input.

## 5. Verifiable Deliverables (selected)

- AI maturity questionnaire & scoring results / radar chart
- Skill Library + Antigravity artifacts
- n8n Workflow PoC + Execution Log
- Hermes Agent task cards + Gate 4A-4E sign-off
- ClawTeam Agent Team role cards and integration report
- Eight-stage consulting diagnostic report + 30/60/90-day Roadmap
