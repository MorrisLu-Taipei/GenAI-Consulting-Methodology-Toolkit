# 05 Sales

This directory holds outward-facing sales materials.

- `SALES_VALUE_PROPOSITION.md`: role-based value propositions, 30-second / 3-minute / 10-minute sales pitches, and common objections.
- `SALES_FAQ.md`: the 20 most-asked sales questions (pricing, timeline, customization, deployment, security, ROI, IP ownership…), bilingual.
- `ONE_PAGER_OUTLINE.md`: content draft + layout brief for a single-page PDF / printed one-pager.
- `EXECUTIVE_DECK_OUTLINE.md`: 10-slide executive deck outline (CEO/COO/CIO — each slide with key message, content, visual recommendation, speaker notes).
- `STANDARD_SALES_DECK_OUTLINE.md`: 20-slide standard sales deck outline (department managers, IT, HR).
- `CONSULTING_METHODOLOGY_DECK_OUTLINE.md`: 30-slide consulting-methodology deck outline (deep methodology, including academic grounding).
- `VISUAL_ASSETS_SPEC.md`: ASCII specs + designer briefs for 3 visual assets (three-step service flow, L1-L5 maturity map, verifiable-deliverables checklist).
