## อย่าพยายามจับทั้งแปดขั้นตอนพร้อมกัน
ในงานครั้งแรกของคุณ จำเพียงลำดับของห้าสิ่งนี้:

1. **วินิจฉัยก่อน** (อย่าออกแบบหลักสูตรหรือเสนอราคาก่อนการวินิจฉัยเด็ดขาด)
2. **จากนั้นออกแบบหลักสูตร** (L1-L5 — เติมช่องว่างที่ขาดหายไปจริง ๆ)
3. **จากนั้นจัดทำรายงาน** (ทุกข้อสรุปต้องมี evidence)
4. **จากนั้นส่งมอบและตรวจรับ** (ส่งมอบ *ความสามารถที่ตรวจรับได้* ไม่ใช่เครื่องมือ)
5. **สุดท้ายเข้าสู่การบำรุงรักษาหรือการทบทวนรายไตรมาส** (ลากเส้นทแยงมุมใหม่ทุกไตรมาส; โครงสร้างดีขึ้นจริงหรือไม่?)

> ห้าขั้นตอนนี้คือโครงกระดูกของแปดขั้นตอน ทำให้โครงกระดูกไหลลื่นก่อน รายละเอียดค่อยว่ากันทีหลัง

## 5 ข้อผิดพลาดที่มือใหม่มักทำ
1. **ขาย 24 เดือนเต็มตั้งแต่แรก** — ขายการวินิจฉัย Phase A ก่อน; ให้ Gate เป็นตัวตัดสินว่าจะไปต่อหรือไม่
2. **ออกแบบหลักสูตรก่อนวินิจฉัย** — หากไม่มีวิจารณญาณ L-level หลักสูตรจะไม่เข้ากันเสมอ
3. **เข้าใจผิดว่าเครื่องมือคือการเปลี่ยนผ่าน** — การนำเครื่องมือมาใช้ ≠ ลูกค้าได้รับความสามารถ; สิ่งที่คุณส่งมอบคือความสามารถ
4. **รายงานที่ไม่มี evidence** — ทุกข้อสรุปต้องชี้ไปยังคำพูดจากการสัมภาษณ์ ข้อมูล หรือการสังเกต
5. **ไม่มี Gate → ขอบเขตงานบานปลาย** — Phase A/B/C ต้องมีจุดออกที่มีการลงนามอนุมัติชัดเจน มิฉะนั้นจะไม่มีวันจบและคุณจะไม่ได้รับเงินงวดสุดท้าย

## Book I ป้อน Book II อย่างไร (เปลี่ยนวิจารณญาณให้เป็นการส่งมอบ)
- **Discovery / สัมภาษณ์**: ใช้ "หาจุดเจ็บปวดที่แท้จริง" และ "📥 input vs 🧠 output" ของ Book I เพื่อดึงความต้องการที่ไม่ได้พูดออกมา
- **Consulting Report**: เขียนเส้นทแยงมุมของ Book I — *จุดบอดของคู่แข่ง × ความต้องการในอนาคตของลูกค้า × จุดเข้าของเรา* — เป็น Executive Problem Statement + Roadmap
- **Delivery / การทบทวนรายไตรมาส**: ลากเส้นทแยงมุมใหม่ทุกไตรมาสเพื่อตรวจสอบว่าโครงสร้างดีขึ้นจริงหรือไม่ — นั่นคือเหตุผลของการต่อสัญญา

## ข้อเตือนด้านความปลอดภัย (ต้องปฏิบัติตาม)
ข้อมูลลูกค้า บันทึกถอดเสียงการสัมภาษณ์ และผลการวินิจฉัย สามารถประมวลผลได้เฉพาะด้วย **AI ที่บริษัทอนุมัติหรือ endpoint ส่วนตัว** เท่านั้น — ห้ามวางลงในบริการสาธารณะที่ไม่ได้รับอนุมัติ ทำให้การอ้างอิงกรณีศึกษาภายนอกทั้งหมดไม่ระบุตัวตน (ใช้ตัวแทน เช่น "Company ABC" / "City X")

> คู่มือนี้เป็นจุดเข้าสู่เครื่องมือ ไม่ใช่ความเห็นทางกฎหมาย / การเงิน / ความปลอดภัยขั้นสุดท้าย ที่ปรึกษาที่เป็นมนุษย์จะตรวจทานก่อนการส่งมอบอย่างเป็นทางการทุกครั้ง
